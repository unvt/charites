開発

ソースコードからビルドする

charitesをソースコードからビルドしてインストールする手順を以下に説明します。

$ git clone https://github.com/unvt/charites.git
$ cd charites
$ npm install
$ npm run build
$ npm install -g .

警告

すでに他のバージョンの charites を npm からグローバルにインストールしている場合、いくつかのコンフリクトが発生する可能性があります。開発版をインストールする前に、以下のコマンドを使用して、グローバルからすべてのcharitesをアンインストールしてください。

npm uninstall -g .
npm uninstall -g @unvt/charites

次に、以下を実行します。

$ charites help

パッケージのリリースをするには

$ npm version patch # Or `minor` or `major`
$ git push
$ git push origin <version>
