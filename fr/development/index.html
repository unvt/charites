Développement

Compiler à partir du code source

Les étapes suivantes indiquent comment complier et installer charites à partir du code source.

$ git clone https://github.com/unvt/charites.git
$ cd charites
$ npm install
$ npm run build
$ npm install -g .

Avertissement

si vous avez déjà installé d’autres versions de charites globalement avec npm, il peut y avoir des conflits. Veuillez désinstaller toutes les versions globales avant d’installer la version de développement.

npm uninstall -g .
npm uninstall -g @unvt/charites

Ensuite, exécutez:

$ charites help

Publier ce package

$ npm version patch # Or `minor` or `major`
